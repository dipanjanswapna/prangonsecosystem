# **App Name**: Prangons Ecosystem

## Core Features:

- Portfolio Showcase: Display Dipanjan's work, skills, and projects in an organized manner.
- Blog and Articles: Publish educational and motivational content written by Dipanjan.
- eBook Library: Host and provide access to a collection of eBooks and resources.
- Service Listings: List the services offered, such as web development and UI/UX design.
- ONGON Donation System: Collect donations for ONGON Bangladesh through integrated payment options. Includes progress bar and donor wall
- Content Generation Tool: AI powered tool to assist Dipanjan in generating outlines and first drafts of blog content and social media posts, based on specified keywords and tones.
- User Authentication and Roles: Enable user sign-up, login, and profile management with role-based access control.

## Style Guidelines:

- Primary color: Deep navy blue (#0A192F) to convey professionalism and depth.
- Background color: Dark gray (#1F2937) complements the primary color and enhances readability.
- Accent color: Neon green (#00FFFF) to highlight key elements and calls to action.
- Headings font: 'Poppins', a geometric sans-serif, for a modern and clean appearance.
- Body font: 'Inter', a grotesque-style sans-serif, for clear and readable content. The other allowed font specified by the user (Siyam Rupali or SolaimanLipi) is currently not supported.
- Implement a mobile-first responsive design with a dashboard-style layout for desktops.
- Use subtle animations and transitions with Framer Motion to create a smooth, app-like user experience.
